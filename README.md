# Mercury_trumpet_effects
SuperCollider script for live trumpet performance effects. 

One mono audio input with a small amount of reverb and delay on it. There is a step sequesncer controlled by pressing 1-8 on a number pad. 3 effects which will be controlled by a light based midi controller; the 1st is a comb delay, the 2nd is a guitar effect, and the 3rd is a synth sound effect. There are also 3 sound controlled parameters - playing a concert F3 will create a rhythmic 16th note chop effect, a G3 will create a  eighth note triplet chop effect, and E3 will create a massive comb delay sweep. Volume controls are "+" and "-" . Mix controls are "/" and "*" . Looper mute is "." Clear looper (step sequencer) is "backspace". All controls are on the number pad for portability. Enjoy.
